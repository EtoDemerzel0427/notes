## 零、历史渊源

历史上是先有“Short”的概念，才有了“Long”。

"Short" 的本意就是 _“短缺” (Shortage/Deficiency)_。在以前的实物交易中，如果你答应卖给别人 100 桶油，但你仓库里其实是空的，按照行话，你就是 _"Caught short"_ (处于短缺状态)。后来投机者发现，如果我预感价格会跌，我可以故意让自己处于这种“短缺”状态（先签卖出合同），等价格跌了再去买便宜货来填补这个“短缺”。所以，Shorting 的本质就是让自己处于负库存（Negative Inventory）的状态。

> 这也许也解释了为什么看跌叫 "Bear" (熊市)。有一句古老的谚语："Don't sell the bear's skin before you've caught the bear"（还没抓到熊，别急着卖皮）。那些卖“空头支票”的人被称为 "Bearskin jobbers"，简称 Bears。

而 "Long" 的出现主要是为了作为 "Short" 的反义词。如果 "Short" 意味着库存是 负数（缺货），那么作为这一状态的对立面，当你实实在在拥有货物，准备长期持有时，你的库存是“长”的（Long supply）。持有实物通常意味着你愿意在这个资产上投入更“长”的时间（Long period of time），等待它增值；而早期的 Short 往往是短期的投机倒把行为。

做空的做法很可能是在 1609 年由荷兰商人 Isaac Le Maire 发明的，他是荷兰东印度公司 (VOC) 的大股东。
此外值得一提的是，早期 hedge fund 的核心创新之一，就是将 long 与 short 结合以对冲风险。

## 一、最直观的理解

### ✅ Long（做多 / 多头）

* **你赌价格会上涨**
* 操作：**低价买入 → 高价卖出**
* 赚钱逻辑：

  > 价格 ↑ → 资产价值 ↑ → 你赚钱

📈 举例：
你觉得 **ES（标普500期货）** 会涨
→ 现在 5000 买入
→ 涨到 5050 卖出
→ **你是 long ES，赚了 50 点**

---

### ❌ Short（做空 / 空头）

* **你赌价格会下跌**
* 操作：**先卖出 → 低价再买回**
* 赚钱逻辑：

  > 价格 ↓ → 回补成本 ↓ → 你赚钱

📉 举例：
你觉得 ES 会跌
→ 现在 5000 “卖出”（借来卖）
→ 跌到 4900 再买回
→ **你是 short ES，赚了 100 点**

⚠️ 注意：

* **做空不是“没买就卖”**
* 本质是：**借资产卖 → 再买回来还**

---

## 二、专业一点的定义（交易员/风控视角）

### Long / Short 描述的是：

> **你对某个风险因子（这里是价格）的方向性暴露**

| 状态    | 对价格变动的敏感性  |
| ----- | ---------- |
| Long  | 价格↑ → PnL↑ |
| Short | 价格↑ → PnL↓ |

数学上（极简）：

```
PnL ≈ position × price_change
```

* `position > 0` → Long
* `position < 0` → Short

## 三、在不同金融工具里的含义

### 1️⃣ 股票（最直观）

| 操作   | 含义       |
| ---- | -------- |
| 买股票  | Long 股票  |
| 融券卖出 | Short 股票 |

> 什么是「融券卖出」（Short Selling / Securities Lending）?\
> 一句话定义：
> 融券卖出 = 向券商借股票 → 卖给市场 → 未来买回来还给券商
> 
> 交易步骤：
> 1. 向券商借股票（融券）：券商从其他客户/机构库存/借券市场借来股票给你
> 2. 你把借来的股票卖出：市场上看到的是一笔普通卖单，但你的账户里：position = -100
> 3. 未来某个时刻你买回来，用来归还借来的股票，这一步叫 cover / buy to cover
---

### 2️⃣ 期货（你熟的，比如 ES / SPX）

| 操作    | 含义       |
| ----- | -------- |
| 买期货合约 | Long 期货  |
| 卖期货合约 | Short 期货 |

👉 期货**天然支持做空**，不需要“借券”这个步骤

---

### 3️⃣ 期权（容易混淆，这里重点说）

⚠️ **期权的 long / short ≠ 看涨 / 看跌**

#### 看四种组合：

| 操作            | 合约方向  | 到期时你希望标的怎么走 | 市场观点         | 最大收益    | 最大风险    |
| ------------- | ----- | ----------- | ------------ | ------- | ------- |
| **Buy Call**  | Long  | 上涨          | **看涨**       | 无限      | 有限（权利金） |
| **Sell Call** | Short | 不涨 / 下跌     | **看跌 / 偏看跌** | 有限（权利金） | **无限**  |
| **Buy Put**   | Long  | 下跌          | **看跌**       | 很大（≈K）  | 有限（权利金） |
| **Sell Put**  | Short | 不跌 / 上涨     | **看涨 / 偏看涨** | 有限（权利金） | 很大（≈K − 权利金）  |


👉 **Long / Short 说的是“合约方向”**
👉 **Bull / Bear 说的是“市场观点”**

例如：

* **Long Put**：你是 *long*，但你是 *bearish*
* **Short Put**：你是 *short*，但你是 *bullish*
