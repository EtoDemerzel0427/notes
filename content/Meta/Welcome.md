# Hello World!

This is my private notebook, hosted on the web using [ RectoWiki](https://github.com/EtoDemerzel0427/RectoWiki). I wrote this Wiki Web app and Desktop app for myself to take notes.

## How it works

1.  I write notes in this repository using [RectoWiki Desktop App](https://github.com/EtoDemerzel0427/RectoWiki/releases).
2.  GitHub Actions automatically pulls the latest commit RectoWiki, and renders the wiki content.
3.  The result is published to GitHub Pages.
